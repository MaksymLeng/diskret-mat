# Discrete Math Quizlet

Проект разбит на нормальные файлы.

## Где лежат данные

Все вопросы, варианты, правильные ответы и объяснения лежат здесь:

```
data/questions.js
```

Формат одного вопроса:

```js
{
  id: "A24-1",
  source: "A-24",
  type: "multi",
  multi: true,
  text: "Текст вопроса",
  options: ["A", "B", "C", "D"],
  answer: [0, 2],
  explain: "Пояснение"
}
```

`answer` хранит индексы правильных ответов с нуля: `0 = A`, `1 = B`, `2 = C`, `3 = D`.

## Где хранится прогресс пользователя

Прогресс не хранится в файлах. Он сохраняется в браузере через `localStorage`.
Ключи:

```
dmQuizProgressSolved_project_v1
dmQuizAnswersSolved_project_v1
```

Кнопка **Сбросить** удаляет эти ключи.

## Как запустить

Вариант 1: просто открой `index.html` двойным кликом.

Вариант 2, если браузер блокирует локальные файлы:

```bash
python3 -m http.server 5173
```

Потом открыть:

```
http://localhost:5173
```

## Что менять

- Вопросы/ответы: `data/questions.js`
- Логика теста: `src/app.js`
- Внешний вид: `src/styles.css`
